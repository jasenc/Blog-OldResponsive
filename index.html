CodePen

May 7th, 2015

CodePen is pretty much the best new thing that I have discovered. The entirety of this blog has been written in it. There are plenty of options to share (make sure you fork a copy or your friends could crash your site...), post to GitHub, look at the source code, and so on. I'm very impressed how well it runs, especially in Internet Explorer 10 on my old refurbished work laptop without admin rights.

I wish I would have looked in to this a lot sooner as I'm sure a lot of topics would have stuck easier if I just played around on CodePen while watching Treehouse or reading some other books. Oh well, at least I still have a ton of JavaScript to work through

First Post

May 6th, 2015

Hi everyone, I've been meaning to start my own blog and have been looking for a way to document all of the things I am learning. I dabbled with HTML & CSS a bit in college over winter break one year but haven't done much since... until now. I started learning again primarily as a hobbie, but quickly became captivated.

I started out looking at various free resources on the internet, trying to put together my own cirriculum, feeling entirely lost for the most part. Finally I landed on Viking Code School, the team over there has put together a great prep section that walks you through what it's like to be a developer, how design is thought of, some of the tools developers use, and finally HTML/CSS and some Ruby. They really cover a lot of material across a lot of different topics. They helped lay down a foundation for me to start building on.

Around the same time I heard about the Code Oregon initiative. The short of it is for any Oregon resident they offer Team Treehouse for free as well as WorkSource for career placement. How could I not take up this oppurtunity? I started with learning Ruby from Tree House as it was the last thing covered by the Viking Code School Prep.

That was fairly simple, I did a lot more programming in college than HTML & CSS. So I went back to HTML & CSS and I've had a difficult time trying to retain the information from all of the videos I have been watching over at Team Treehouse, along with the fact that each day I get distracted about some other new and exciting thing outside of or later on in the Front-End Developer track on Team Treehouse. To help along with that I bought Jon Duckett's Web Design books which have been great. I've also found tons of useful information on GitHub and worked through Shay Howe's Learn to Code HTML & CSS

So as you've just read, I've done quite a bit of learning, now it's time to start applying, which is one reason for this blog...
